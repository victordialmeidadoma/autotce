⚡ SETUP RÁPIDO (5 PASSOS)

PASSO 1: Organize os arquivos
   └─ Copie os 22 arquivos para as pastas corretas

PASSO 2: Crie .env.local
   └─ cp .env.example .env.local
   └─ Preencha com credenciais Supabase

PASSO 3: Instale dependências
   └─ npm install
   └─ pip install -r requirements.txt

PASSO 4: Configure banco de dados
   └─ Vá em Supabase → SQL Editor
   └─ Cole o conteúdo de: SQL_CRIAR_TABELAS_DOCUMENTOS.sql
   └─ Clique Run

PASSO 5: Teste
   └─ npm run dev
   └─ Abra: http://localhost:3000
