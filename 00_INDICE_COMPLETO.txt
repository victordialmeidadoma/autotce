🚀 PASSO-A-PASSO RÁPIDO

1. Criar pastas:
   mkdir seu-projeto-tce
   cd seu-projeto-tce
   mkdir -p api pages/api components styles public

2. Copiar TODOS os 22 arquivos para as pastas corretas (conforme lista acima)

3. Renomear arquivos (remover prefixo):
   - pages_index.jsx → index.jsx
   - pages__app.jsx → _app.jsx
   - pages_api_*.js → *.js
   - components_*.jsx → *.jsx
   - styles_*.css → *.css
   - api_*.py → *.py

4. Instalar dependências:
   npm install
   pip install -r requirements.txt

5. Criar .env.local:
   cp .env.example .env.local
   (Edite com suas credenciais Supabase)

6. Criar banco de dados:
   Supabase → SQL Editor
   Cole: SQL_CRIAR_TABELAS_DOCUMENTOS.sql
   Run ✅

7. Testar:
   npm run dev
   (Abra http://localhost:3000)

8. Deploy:
   git add .
   git commit -m "Sistema completo Vercel + Scraper"
   git push
   (Vercel faz deploy automático)
